Lesson 6- Embeds

Embedding images and iframes are an important part of showing the content you want on your webpage.

Images

There are two ways to embed images. Local images, which are images that are saved in website and can be accessed via file paths, or remote images, which link directly to the image through a link address. The path for both are:

Local image

<img src="checkmark.png" alt="check_mark">

Remote image

<img src = "https://example.com/checkmark.png" alt="check_mark">

<img> declares that an image is going to be embeded, src is what locates the image, and alt describes the image for screen readers or crawlers.

Iframes

Iframes are used one HTML document to display the contents of another. This is achieved with the <iframe> tag. The standard structure for it is "<iframe src=”url” title=”description”></iframe>". This will display another webpage in a small box on your webpage.
