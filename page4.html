Lesson 4- Attributes

Attributes are essentially tag modifiers. They are useful in having one function be changed beneficially, such as using the style attribute to size an image.

Common attributes

  1. style- Lets the user style photos and content using CSS declarations.
  2. src- Locates images when attatching one.
  3. href- Locates links when navigating to a page.
  4. alt- Gives a description of an image if it cannot be shown.
  5. download- Lets the user download a file.
