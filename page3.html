Lesson 3- Common Tagging

HTML has some tags that are frequently used, and knowing them is essential. Some are block level, some are inline. What's the difference?

Block level elements vs. inline elements

Block level elements take up an entire line, and cannot be put next to each other. Some examples are <head>, <p>, <h1>, and more. Inline elements rather, can be placed next to each other on the same line. Some examples are <strong>and <i>.

Headings

Headings are block level elements and are headings for a page ordered from 1-6, ranging in size, with <h1> being the largest. Typically you would want to use the headers in order from h1-h6, often not going past h3 or h4.

Comments

Comments are writing inside the code that are only visible to those who intentionally look at the code. They are used to provide reminders, insight, or any message to one's self or to others reading. The format for a comment is "<!--" before the comment and "-->" after it. There's a comment in this page's code.
