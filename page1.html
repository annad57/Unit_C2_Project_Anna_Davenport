Lesson 1- Fundamentals of HTML

To write an HTML document, there are certain core tags that make up the bare bones of the code. Those are, <! DOCTYPE html>, <html>, <head>, and <body>. When writing HTML, you must make sure that these elements are all present in your code. Starting from Doctype to html to head to body.

  • <! DOCTYPE html>- Declares the document as being written in HTML.
  • <html>- Begins and ends the page
  • <head>- Holds metadata (important information about the webpage).
  • <body>- Contains the visible content (content the viewer can see) on the web page.
