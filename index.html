Let's Learn HTML!

This webpage is a guide to basic HTML explained in a few lessons.

What is HTML?

HTML stands for HpyerText Markup Language. It is a coding language that is written through tags that have different functions. It is used to markup document webpages and are the basis of their content. Tags are written standardly with a starting bracket (<example>) and an ending bracket (</example>) .

How do you start writing HTML?

There are lots of different places to write HTML, though the easiest place to start is an online editor, such as StackBlitz. That is what this website is coded on!
