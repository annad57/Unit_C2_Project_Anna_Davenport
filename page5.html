Lesson 5- Formatting Text

Tables

Tables are easy to make in HTML. They consist of 4 tags:

  1. table- Creates a table
  2. tr- Creates a row
  3. th- Creates a heading cell
  4. td- Creates a data (column) cell

The attributes colspan and rowspan are also used, as they merge cells vertically or horizontally.


Write the table declaration first, then start with a row and insert heading or column cells left to right. Here's an example table:

x y z
1 2 6 5
9 2 3 4
7 8 1

Lists

Lists are also very easy to create in HTML, and consist of three tags:

  1. ol- Creates an ordered list
  2. ul- Creates an unordered list
  3. li- Creates an item for the list

Start with putting either an ol or ul tag, and then li for every list item. Here's two example lists:

  1. This
  2. Is
  3. A
  4. Numbered
  5. List
  • This
  • Is
  • An
  • Unnumbered
  • List

<pre> Tag

HTML ignores any unnecessary spaces or line breaks in your paragraphs. This can be annoying when spaces and breaks are necessary. There are multiple ways to solve this issue. Firstly, we can use <br> or empty paragraph tags for spaces and lines. Or, we use the <pre> tag. This tag forces the page to display the text as it is written, with spaces and coding characters, removing the need to always use the first option for formatting. Here's an example:

This    text
is      very


    Cool

All spaces and line breaks are preserved!
